## 变量
### 变量的分类
全局变量在大括号的外边，而局部变量在大括号内部。
当全局变量和局部变量`变量名相同`时，`优先`使用`局部变量`，且两个是`不同`的变量。

### 变量的使用
作用域：一个变量哪里可以使用哪里就是他的作用域。
生命周期:从变量的生产到销毁的过程即为变量的生命周期
局部变量：
作用域：大括号内部
生命周期：进入局部到出局部的范围
全局变量：
作用域：整个工程
生命周期：整个程序的生命周期

## 常量
### 字面常量
如“20”整形常量，“3.14”浮点型常量，“a”,字符常量.......
 ### const 修饰的常变量
const 是C语言的关键字
```
const int a = 10; #a即为一个常变量，但其本质仍为变量

int arr[a]={0} #会报错，说明a是一个变量

```
### #define 定义的标识符常量
在main（）函数外用#define定义
```
#define N 100 #此时N即为标识符常量

int arr[N]={0} #不会报错，说明N此时是常量
```
### 枚举常量
```
enum 关键字，枚举关键字

格式：
enum 枚举名
{
	枚举常量,
	枚举常量, //用逗号（，）隔开
	枚举常量  //最后一个没有，
}；//大括号外的；不可省略

例如：
enum SEX
{
	Male,
	FEMALE,	
	SECRET
};
//且用%d 打印枚举常量时，以 0开始，与数组相同
printf("%d",Male) //  0
```
## 字符与字符串
多个字符拼接在一起，用双引号引用，即为字符串常量，且以`‘\0’`,作为结尾。
### strlen（）
 统计字符串的长度，在`\0之前`的字符串长度。
```
char arr1[]="abc"
char arr2[]={'a','b','c'}
printf("%s",arr1) //abc
printf("%s",arr1) //abc后乱码随机值
//arr2[]={'a','b','c','\0'}则arr1一样

```
